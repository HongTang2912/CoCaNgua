RUN PROJECT:
1. Download all file in folder GamePy by cloning from url https://github.com/HongTang2912/CoCaNgua
2. Go to GamePy/dist/ and open gamefinal.py file to run project.
3. Press 2, 3 or 4 for number of players, maximum 4 members.
4. How to play: 
  - Red players First, press Space to shake dices, 1 to out chess, 2 to choose a chess on play board, 3 and 4 to move lean on the dices unit.
  - Who get 4 chesses returned home of the board will be the winner.
